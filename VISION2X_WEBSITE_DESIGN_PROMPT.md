# Vision2X Website Design & Development Prompt

## Project Overview
Create a modern e-commerce website for Vision2X (vision2x.com) - a company specializing in AI-powered perception systems for autonomous vehicles, fleet safety, and micromobility. The website should function as a clean online store with quick purchasing capabilities, order tracking, and comprehensive product information.

## Brand Identity
- **Company**: Vision2X Technologies
- **Domain**: vision2x.com
- **Tagline**: "Empowering Tomorrow's Mobility"
- **Color Palette**: 
  - Primary: Deep Blue (#0A2342)
  - Secondary: Electric Cyan (#00D4FF)
  - Accent: NVIDIA Green (#76B900)
  - Success: Bright Green (#00FF88)
- **Typography**: Modern sans-serif (Inter, Roboto)
- **Visual Style**: Clean, professional, futuristic, trustworthy

## Products Portfolio

### 1. VisionSense
- **Target**: Researchers and developers in autonomous vehicle technology
- **Variants**: 
  - 8GB Jetson Orin Nano version ($2,999)
  - 16GB Jetson Orin Nano version ($3,999)
- **Key Features**: Stereo + mono cameras, AutoVision software, ROS2 compatible
- **Use Cases**: University research, AV development, perception algorithm testing

### 2. VisionConnect
- **Target**: Fleet operators and commercial vehicle companies
- **Variants**:
  - Basic Safety Suite ($1,499)
  - Safety + C-V2X Module ($1,999)
  - Complete Fleet Package ($2,499)
- **Key Features**: V2X communication, OBU connectivity, fleet dashboard, driver monitoring
- **Use Cases**: Trucking fleets, delivery vehicles, public transportation

### 3. VisionRide
- **Target**: Micromobility operators and manufacturers
- **Variants**:
  - VisionRide Lite ($599)
  - VisionRide Pro ($899)
  - VisionRide Fleet ($1,199)
- **Key Features**: Compact design, obstacle detection, rider alerts, fleet analytics
- **Use Cases**: E-scooters, e-bikes, shared mobility services

### 4. AutoVision Software
- **Type**: Open-source perception stack
- **Features**: Pre-trained AI models, ROS2 native, TensorRT optimized
- **Distribution**: Pre-installed on all hardware, GitHub repository available
